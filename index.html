Cheat Sheet de Python

Diccionarios

1 / 16

¿Qué es un Diccionario?

Una colección no ordenada de pares clave: valor. Son modificables y no permiten claves duplicadas.

2 / 16

Crear un Diccionario

Se define con llaves {}, con cada par separado por dos puntos.

persona = {
    "nombre": "Ana",
    "edad": 25,
    "ciudad": "Madrid"
}
3 / 16

Acceder a un Valor

Se accede al valor usando su clave entre corchetes.

persona = {"nombre": "Ana", "edad": 25}
print(persona["nombre"]) # 'Ana'
4 / 16

Acceso Seguro: .get()

Devuelve None (o un valor por defecto) si la clave no existe, evitando errores.

persona = {"nombre": "Ana"}
print(persona.get("edad")) # None
print(persona.get("edad", 18)) # 18 (valor por defecto)
5 / 16

Modificar un Valor

Se reasigna un nuevo valor a una clave existente.

persona = {"nombre": "Ana", "edad": 25}
persona["edad"] = 26
print(persona) # {'nombre': 'Ana', 'edad': 26}
6 / 16

Añadir un Par Clave-Valor

Se asigna un valor a una clave que no existe en el diccionario.

persona = {"nombre": "Ana"}
persona["profesion"] = "Ingeniera"
print(persona)
7 / 16

Eliminar: del

Elimina un par clave-valor usando la palabra clave `del`.

persona = {"nombre": "Ana", "edad": 25}
del persona["edad"]
print(persona) # {'nombre': 'Ana'}
8 / 16

Eliminar: .pop()

Elimina un par y devuelve el valor eliminado.

persona = {"nombre": "Ana", "edad": 25}
edad = persona.pop("edad")
print(edad) # 25
print(persona) # {'nombre': 'Ana'}
9 / 16

Obtener Claves: .keys()

Devuelve un objeto iterable con todas las claves del diccionario.

persona = {"nombre": "Ana", "edad": 25}
print(persona.keys()) # dict_keys(['nombre', 'edad'])
10 / 16

Obtener Valores: .values()

Devuelve un objeto iterable con todos los valores.

persona = {"nombre": "Ana", "edad": 25}
print(persona.values()) # dict_values(['Ana', 25])
11 / 16

Obtener Items: .items()

Devuelve un objeto iterable con pares (clave, valor) como tuplas.

persona = {"nombre": "Ana", "edad": 25}
print(persona.items())
# dict_items([('nombre', 'Ana'), ('edad', 25)])
12 / 16

Recorrer Claves (Loop)

Por defecto, un `for` sobre un diccionario recorre sus claves.

for clave in persona:
    print(clave) # nombre, edad
13 / 16

Recorrer Items (Loop)

La forma más común de recorrer un diccionario para obtener clave y valor.

for clave, valor in persona.items():
    print(f"{clave}: {valor}")
14 / 16

Operador in

Verifica si una clave existe en el diccionario. Es muy rápido.

if "nombre" in persona:
    print("La clave 'nombre' existe.")
15 / 16

Función len()

Devuelve el número de pares clave-valor.

print(len(persona)) # 2
